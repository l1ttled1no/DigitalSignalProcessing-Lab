# Digital Signal Processing Lab

Source code for the Digital Signal Processing laboratory assignments of the course **CO2035 - Digital Signal Processing**, Semester 252, at Ho Chi Minh University of Technology (HCMUT).

All scripts are implemented in **Scilab**, a free and open-source platform for numerical computation.

## Repository Structure

- `src/<lab_name>/`: Scilab source files for each lab.
- `doc/<lab_name>.pdf`: Corresponding lab reports.
